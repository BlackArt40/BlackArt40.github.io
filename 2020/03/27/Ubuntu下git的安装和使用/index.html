Ubuntu下git的安装和使用

安装git

  1. 在终端中输入git可检查是否已经安装，若没安装可以根据提示的命令行进行安装

    $ git

    ps:很多软件都可以通过这种方式安装。

  2. 根据提示命令进行安装

    $ sudo apt install git

  3. 可能出现（没有可用的软件包git，但是它被其他的软件包引用了）错误

    这可能意味着这个缺失的软件包可能已被废弃，或者只能在其他发布源中找到

  4. 此时只需要更新一下源

    $ sudo apt-get update

  5. 待源更新完毕之后再次输入第二步的安装命令

    $ sudo apt install git

  6. 等待安装完毕之后再次输入git，就会发现已经成功安装

  1. 安装完成后，进行最后一步设置，在命令行输入：

    $ git config --global user.name "Your Name"

    $ git config --global user.email "email@example.com"
